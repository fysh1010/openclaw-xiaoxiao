# 📋 更新日志

## 2026-03-20 - 界面优化升级

### ✨ 新增功能

#### CSS 动效系统
- 新增 `gradientShift` 渐变动画
- 新增 `pulse` 脉冲动画
- 新增 `slideInUp` 滑入动画
- 新增 `fadeIn` 淡入动画
- 新增 `glow` 光晕动画
- 新增 `spin` 旋转加载动画
- 新增 `shimmer` 骨架屏闪烁效果

#### 高级视觉组件
- `.hero-enhanced` - 增强版 Hero 渐变背景
- `.hero-decoration` - 浮动装饰元素
- `.glass-card` - 玻璃态卡片效果
- `.gradient-border` - 渐变边框效果
- `.btn-glow` - 发光按钮效果
- `.loading-spinner` - 加载旋转器
- `.skeleton` - 骨架屏占位

#### 实用 UI 组件
- `.tag` - 标签样式系统
- `.progress-bar` - 进度条
- `.badge` - 通知徽章
- `.tooltip` - 悬浮提示
- `.diary-timeline` - 日记时间线布局

#### 响应式支持
- 深色模式 `@media (prefers-color-scheme: dark)`
- 打印样式 `@media print`
